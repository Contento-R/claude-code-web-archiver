# Claude Code Web Session Archiver

> Сохраняет **всю** сессию Claude Code Web — включая историю, которую облако уже сжало, — в один самодостаточный HTML-файл (или Markdown / JSON) со встроенными скриншотами как data-URL.

Userscript для Tampermonkey / Violentmonkey / Greasemonkey, который добавляет на [code.claude.com](https://code.claude.com), [claude.ai/code](https://claude.ai/code) и [claude.com/code](https://claude.com/code) **сворачивающуюся** маленькую зелёную панель с пятью элементами управления:

| Кнопка          | Что делает                                                                                                          |
| --------------- | ------------------------------------------------------------------------------------------------------------------- |
| ⬇ **Архив**     | Пройти по всей сессии, захватить каждое сообщение, скачать скриншоты, сохранить один автономный файл.              |
| ⚡ **Быстро**    | Переключатель. Агрессивные задержки и 10 параллельных загрузок скриншотов вместо 6.                                 |
| 📝 **Без кода** | Переключатель. Исключает блоки кода (tool-вызовы, диффы, файл-вьюверы, `<pre>`) — только переписка.                |
| ⚙ **Настройки**| Формат вывода (HTML/MD/JSON), only-new, range, локальная сеть, редакция секретов, сворачивание ответов Claude.      |

Полоска `⋮⋮` слева — drag-хэндл; позиция сохраняется в `localStorage` и автоматически прижимается к viewport при resize, так что панель не уходит за пределы экрана.

Через **3 секунды** после ухода курсора / фокуса панель сворачивается в маленький зелёный круг. Hover или фокус — разворачивается обратно. Сворачивание приостановлено во время архивации и при открытой модалке настроек. Круг тоже перетаскивается.

**Хоткеи:** `Alt+A` — старт, `Esc` — отмена.

UI на **EN / RU / DE / FR / ES** автоматически по `navigator.language` (отсутствующие переводы прозрачно падают на английский).

## Зачем

Claude Code Web сжимает длинные сессии. Старые сообщения исчезают из живой ленты и остаются только в виде краткого пересказа. Этот скрипт обходит живой DOM, пока всё ещё отрисовано, и сохраняет полный транскрипт (со скриншотами) до следующего сжатия.

## Установка

1. Установите менеджер userscript-ов:
   - [Tampermonkey](https://www.tampermonkey.net/) (Chrome, Edge, Firefox, Safari, Opera)
   - [Violentmonkey](https://violentmonkey.github.io/) (Chromium / Firefox)
   - [Greasemonkey](https://www.greasespot.net/) (Firefox)
2. Откройте сырой файл скрипта — менеджер сам подхватит его:
   [`claude-code-web-archiver.user.js`](https://github.com/Contento-R/claude-code-web-archiver/raw/main/claude-code-web-archiver.user.js)
3. Подтвердите установку. Дальше обновления подтягиваются автоматически через `@updateURL`.
4. Откройте любую сессию по адресу `https://code.claude.com/*`, `https://claude.ai/code/*` или `https://claude.com/code/*`. В правом нижнем углу появится зелёная панель.

## Использование

1. Откройте сессию Claude Code Web.
2. (Опционально) Нажмите ⚙, чтобы выбрать формат вывода и включить нужные опции.
3. Нажмите **⬇ Архив** (или `Alt+A`) и подтвердите запрос.
4. **Не трогайте страницу**, пока скрипт работает — взаимодействие сбивает авто-прокрутку и захват.
5. Следите за прогрессом в плашке:
   - `Прокрутка и захват… сообщений: N`
   - `Скачиваю скриншоты… X/Y (Z ok)`
   - `Собираю HTML…`
   - `Готово! Сообщений: N, картинок встроено: M.`
6. Браузер сохранит файл `<название-сессии>.html` (или `.md` / `.json`).

Прервать — кнопка **Отмена** в плашке прогресса или `Esc`.

### Форматы вывода

Выбирается в **Настройках → Формат вывода**:

- **HTML** — интерактивный самодостаточный файл. Включает live-поиск, dropdown «Перейти к сообщению», переключатель тем (тёмная/светлая), кнопку свернуть/развернуть все ответы, печатный CSS.
- **Markdown** — чистый текст с заголовками, списками, fenced code blocks, blockquotes, таблицами, ссылками и data-URL картинками. Для Obsidian, GitHub и т.п.
- **JSON** — структурированный документ с per-message `number / role / tool / time / text / html` и session-level `model`. Для машинной обработки.

### Режим ускорения (⚡)

|                          | Обычный | Быстрый |
| ------------------------ | ------: | ------: |
| Пауза прокрутки (мс)     |     500 |     140 |
| Пауза разворачивания (мс)|      90 |      20 |
| Шаг прокрутки (vh)       |    0.7  |     0.9 |
| Шагов без прогресса      |       4 |       3 |
| Параллельных загрузок    |       6 |      10 |

### Режим «Без кода» (📝)

Удаляет блоки кода, которые пишет Claude — tool-вызовы (Bash, Edit, Write…), `<pre>`-блоки, файл-вьюверы, моноширинные контейнеры. Инлайновый `code` внутри текста сохраняется. Детект идёт по computed `font-family` живого DOM, независимо от классов UI.

### Настройки

- **Only new** — пропускать сообщения, уже захваченные в предыдущих архивах этого URL. Per-URL known-keys, до 5000.
- **Range from/to** — выгружать только сообщения с номерами в этом диапазоне.
- **Локальный режим сети** — не использовать cross-origin `GM_xmlhttpRequest` fallback. Картинки, которым он нужен, остаются с remote URL.
- **Скрывать секреты** — список regex через newline. Дефолты покрывают ключи OpenAI/Anthropic, GitHub PAT, AWS-ключи. Каждое совпадение заменяется на звёздочки на стадии сборки HTML, через `TreeWalker` — без риска повредить разметку.
- **Сворачивать ответы Claude** — оборачивать тела ассистентских сообщений в `<details>`, чтобы экспортированный HTML открывался компактно.

### Tool & metadata badges

В выводе у каждого сообщения возле роли добавляются опциональные бейджи:
- **Tool-call** (Bash / Edit / Write / Read / Glob / Grep / WebFetch / WebSearch / Task / TodoWrite / NotebookEdit / NotebookRead / MultiEdit / ExitPlanMode / SlashCommand / KillShell).
- **Модель** (Opus / Sonnet / Haiku, вытащено из chrome страницы).
- **Время** (`<time datetime>` или `[title]` с date/time паттерном; формат `YYYY-MM-DD HH:MM`).

### Resume после краха

Если вкладка закрылась / обновилась / архив прерван в середине прокрутки, следующий клик Archive для того же URL предложит продолжить с последнего снимка (snapshot каждые ~5 секунд во время скролла, TTL 24 часа).

### Уведомление об обновлении

Скрипт раз в сутки проверяет `main` на GitHub. При новой версии — жёлтая точка на кнопке ⚙ и плашка с «Установить» в верхней части модалки настроек. По клику открывается сырой URL — Tampermonkey предложит штатную переустановку.

## Права / сеть

Скрипт запрашивает `GM_xmlhttpRequest` и `@connect` к:

- `code.claude.com`, `claude.ai`, `claude.com`, `anthropic.com` — контейнеры сообщений и вложения
- `cloudfront.net`, `amazonaws.com` — CDN для картинок
- `raw.githubusercontent.com` — раз в сутки, прочитать `@version` опубликованной версии
- `self` — same-origin запросы

Никаких данных никуда не отправляется, кроме GitHub raw URL выше — для проверки обновлений.
